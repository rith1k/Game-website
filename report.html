Website Report

My website

Home Page

The Home page is the index page of the website, which lays out the theme of the entire website with the help of animations and gif backgrounds.

Login/Sign-Up:

The login page contains two forms, one for signing up and one for logging in. The users will be redirected to the game page when the login is successful.

Game Page

The Game page is the main page of the website where users would be able to play the game, identify the objects and score points to get featured in the leaderboard!

About-Us Page

The About-Us page contains details about the game, and how to play it.

Leaderboard Page

The Leaderboard page contains the score of the top 10 users who beat the level in the shortest amount of time.

HTML tags

Stucture Elements:

Structure tags are those tags that help in formating the structure of the contents in a website. The structure elements used in this website are: Nav, Article, Section, Header, Footer.

nav:

I used the nav function to link all my webpages to one another.

header:

I used the header tag to contain the navigation bar to make it appear on top of my website.

article:

The article tag was used in my About-us page to mainly for its self contained composition.

section:

The section tag was used to group together similar li's in my website and provide a typewriter effect using css.

footer:

The footer tag contains the copyright information about my website and sticks at the bottom

Content Elements:

Content tags are all those tags that help inputting any form of content into the website. The content tags used in this website are: a, input, form, p, ul, li, img, div, h1-h6, table, &'copy .

a

a tag was used to link from the about-us webpage to the leaderboard webpage.

Group of html tags.

form

Form tag was used to create a signup and login form for users to register/login.

input

Input was used for the login and signup page to allow users to input information such as username, password,email etc.

p

p tag was used to write a block of content on the index page of the website.

ul

ul tag was used in the aboutpage to create a list of unordered items.

li

li tag was used for providing a custom image for each list-item.

img

img tag was used for inserting ../images into my website.

div

div tag was used to group multiple elements together to provide a mutual style to all of those elements.

h1-h6

h1 tag was used to make a heading on the index page of the website.

table

The table tag was used in the leaderboard webpage which had 10 rows and 4 columns.

&'copy

&'copy tag was used to embed a copyright symbol(©) in the footer of the website.

The outputBannerNavigation function contains the header of the webpage, which includes the navigation bar for all of the website. The function helps in highlighting the selected navigation element of the navigation bar by using the selected class.

outputFooter:

The outputFooter function contained the footer of the webpages and had the closing body and html tags.
